# Project 1 – Retail Sales Analysis

## Overview
This project analyses a retail sales dataset using SQL to answer business questions and provide actionable insights.

## Skills Demonstrated
- SQL
- GROUP BY
- SUM()
- AVG()
- COUNT()
- HAVING
- ORDER BY
- LIMIT
- Business Analysis
- Data Interpretation

## Business Questions Answered
- What is the total revenue?
- What is the average transaction value?
- Which product category generated the highest revenue?
- Which gender generated the highest revenue?
- Which age group spends the most?
- Which ages generate the most revenue?

## Key Findings
- Highest Revenue Category: Electronics
- Lowest Revenue Category: Beauty
- Highest Average Transaction Value Category: Beauty
- Most Transactions Category: Clothing
- Highest Revenue Gender: Female
- Highest Average Transaction Value Gender: Female
- Highest Revenue Age: 43
- Highest Average Transaction Value Age: 37 (£728.125)
