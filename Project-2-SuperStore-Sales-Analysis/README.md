# Project 2: SuperStore Sales Analysis

## Project Overview

This project analyses SuperStore sales data using SQL to answer business questions about sales, profit, products, orders and shipping performance.

## Tools Used

- SQL
- SQLite
- DB Browser for SQLite

## SQL Skills Used

- SUM, AVG and COUNT
- GROUP BY
- ORDER BY
- LIMIT
- CASE WHEN
- Subqueries
- Date functions using strftime()
- Date difference using julianday()

## Business Questions Answered

- Which year-month had the highest total sales?
- What was the total profit for each year?
- How many days did each order take to ship?
- Which orders had sales above the overall average?
- Which products had profit above the overall average?
- Which year-month had the highest total profit?

## Key Learning

This project helped me practise turning business questions into SQL queries and using date functions, subqueries and aggregation to analyse business performance.
